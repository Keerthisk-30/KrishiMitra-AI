# KrishiMitra AI 🌾

KrishiMitra AI is an AI-Powered Farmer-to-Farmer Knowledge Marketplace developed using Python Flask, MySQL, Bootstrap, JavaScript, and AI/NLP technologies.

The platform helps farmers share agricultural problems, access multilingual farming guidance, receive AI chatbot support, weather advisories, analytics dashboards, voice assistance, and smart farming recommendations.

# 🚀 Features

- AI Farming Chatbot
- Weather Advisory System
- Multilingual Support
- AI Text Summarization
- Voice Assistance
- Analytics Dashboard
- Farmer Post Sharing Platform
- Secure Login & Registration

# ▶️ How to Run

1. Clone Repository
git clone https://github.com/Keerthisk-30/KrishiMitra-AI
2. Install Dependencies
pip install -r requirements.txt
3. Run Flask Application
python app.py
4. Open Browser
http://127.0.0.1:5000
